# Movie Streaming Front End

A react frontend for a video streaming site.

## Running

```bash
# Development server
npm run dev
```

Or from the /dist directory

```bash
# Static server (npm)
npm install -g serve
serve -s -l 3000

# Static server (Python)
python -m http.server 3000

# Static server (miniserve)
miniserve-[OS] --index.html --port 3000
```
